EDULOD Educational tools for Linked Open Data

EDULOD is a project at the Institute for ICT-Based Management of the Bern University of Applied Sciences. Read the project proposal for more information

The project is funded by the Hasler Foundation.

LodIDE

LodIDE is the online development environment to easily get started using Linked Open Data (LOD) developed as main result of the EDULOD project.
